# ESP32-S3-WROOM-1-Stamp_Board
A custom ESP32-S3-WROOM-1 compatible stamp-board designed in KiCad using the ESP32-S3 SoC. Unlike the original ESP32-S3-WROOM-1, this design replaces the PCB antenna with a 2.4 GHz chip antenna, along with an RF matching network, external QSPI flash, 40 MHz crystal oscillator and castellated edge pads for seamless integration into embedded systems.
